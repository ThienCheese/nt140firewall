Thiết lập Client ("Một nút bấm")

Sử dụng các phương pháp này để kết nối thiết bị của bạn với tường lửa.

  • iOS & macOS: Tải Hồ sơ Cấu hình DoH
  • Windows(Kết nối trong mạng LAN): Tải connect.ps1 và disconnect.ps1. (Nhấp chuột phải -> Run with PowerShell)
  • Windows (Kết nối từ xa): Tải connect_doh.ps1 và disconnect_doh.ps1. (Nhấp chuột phải -> Run with PowerShell)
  • Linux: Tải connect.sh và disconnect.sh. (Chạy bằng `sudo`)
  • Android (9+): Vào Cài đặt -> Mạng -> DNS cá nhân. Nhập: nt140firewall.duckdns.org
